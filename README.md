## Quick Setup

Install the module to your Nuxt application with one command:

```bash
npm i @anweb/nuxt-ancore
```

Then, add it to your nuxt.config.ts:
```ts
export default defineNuxtConfig({
   modules: ['@anweb/nuxt-ancore']
})
```
